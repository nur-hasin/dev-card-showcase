# Scent Memory Logger

A web app where users log memorable scents and their associated emotions, building a personal scent-memory map.

## Features
- Add, view, and remove scent memories
- Tag memories with emotions and locations
- Visualize scent memories by emotion and location
- Personal scent-memory map and dashboard

## Usage
Open index.html in your browser. Add new scent memories, explore by emotion or location, and reflect on your personal scent-memory map.
