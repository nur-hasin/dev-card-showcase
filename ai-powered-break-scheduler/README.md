# AI-Powered Break Scheduler

A web app that analyzes user work patterns and automatically schedules microbreaks, adapting to deadlines and energy levels.

## Features
- Tracks work sessions and break times
- Adapts break schedule to deadlines and energy
- AI analysis of work patterns
- Persistent data storage
- Responsive UI

## Usage
Open index.html in your browser and start tracking your work. Add deadlines and monitor energy levels for optimal productivity.
