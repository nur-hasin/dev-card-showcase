# Compliment Chain Reaction

A gamified app where each compliment sent unlocks the ability for the recipient to send another, creating a chain of kindness.

## Features
- Send compliments to unlock chain reactions
- Only recipients of compliments can send the next one
- Chain and feed visualization
- Local storage for compliments

## Usage
Open index.html in your browser. Send compliments, unlock new links in the chain, and watch kindness spread!
