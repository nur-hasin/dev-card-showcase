# Mindful Notification Filter

An extension that filters notifications based on user stress and focus levels, only allowing important alerts during deep work sessions.

## Features
- Filters notifications by stress and focus
- Only important alerts during deep work
- Customizable settings in popup
- Visual lists of allowed and filtered notifications

## Usage
Install the extension, open the popup, and set your focus and stress levels. Only important notifications will be allowed during deep work or high stress.
